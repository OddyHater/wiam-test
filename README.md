Проект тестового задания для Wiam
[Задание](https://drive.google.com/file/d/15iUFvEGMVJJaln9RK0tHnyYj5Ii9ZQ8o/view)

Запуск проекта:
1. Клонируйте репозиторий: `git clone https://github.com/OddyHater/wiam-test.git`
2. `npm install`
3. `npm run dev`
4. Проект запущен на `localhost:5173`

[Деплой](https://wiam-test.vercel.app/)

Затраченое время ~7 часов

##Библиотеки и фреймворки:

- mantine 
  - Удобное использование компонентов, без сильного изменений конфига проекта (в отличии от shadcn)
  - Select и Inputs без танцев с бубном
  - Хорошая читаемость
- zod
  - Для валидации в связке с react-hook-form
- tailwind
  - Удобный способ быстро стилизовать ui
- react-hook-form
  - Удобный способ работы с формами
  - База
